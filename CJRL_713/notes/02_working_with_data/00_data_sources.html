Searching for data

Search by file type by adding to your search

  • look only for spreadsheets filetype:XLS filetype:CSV
  • look for geodata filetype:shp
  • look for database extracts filetype:MDB, filetype:SQL, filetype:DB
  • look for PDF files filetype:pdf

Search by part of a URL

  • inurl:downloads filetype:xls will try to find all Excel files that have “downloads” in their web address
  • site:agency.gov to limit your search to only those results on a single domain name

Search for bulk data

  • site:agency.gov Directory Listing may give you some listings generated by the web server with easy access to raw files
