Functions

What is a function?

  • Goal is to create new information from existing data
  • Functions performs operations on text + numbers
  • For example
    • Numbers:
      • Calculating averages, sums (totals)
      • Finding median, min, max, counts
      • Rounding numbers, calculating time/ranges
    • Strings (text):
      • splitting data into multiple columns (YY-MM-DD to year, month, day),
      • or concatenating (joining) text (city, province, country into one column)

Some common functions

Look at some of these functions in action:

Getting data
-Return x character(s) from the left or right of specific cell.
=LEFT(A2,1)
=RIGHT(A2,1)

Joining data

  • Join text together (each cell separated by the delimiter, "-"):
    =JOIN("-",D2,E2,F2)

  • Concatenate/join text together (separated by specific text)
    =CONCATENATE=(D2," and ", E2, "will be joined together with pal", F2)

Basic math

  • Find sum (total) for specific values:
    =SUM(A2, B2, C2)

  • Find sum (total) for a range of values:
    =SUM(A2:A100)

  • Find the average of a specific values:
    =AVERAGE(C2,C5,C10)

  • Find average of a range of values:
    =AVERAGE(C2:C100)

  • Find the median value in a set of values

  • Median: when a set of values is sorted,
    =MEDIAN(C1:C101)

  • Find the highest value(s)
    =MAX(C1:C101)

  • Find the lowest value(s)
    =MIN(C1:C101)

  • Do some simple math (addition, subtraction, multiplication)
    =(C1 / C101)
    =(C1 * C101)
    =(C101 - C1)
    =(C101 + C1)

  • Calculate a date range
    =DATEIF(136)

Checking data

  • Checks one value if a logical expression is TRUE and another if it is FALSE:
    =IF(A2 = "sunshine", "True: Nice weather", "False: Lousy weather")
    =IF(A2 >= 12, 1, 0)
    =IF( (A2 / 10 > 100), 1, 0)

  • Returns the position at which a string is first found within text, case-sensitive:
    =FIND("wood","How much wood can a woodchuck chuck")

  • How it works:
  • =FIND: name of the function
  • (search_for, text_to_search, [starting_at]): inside the round brackets, we have two mandatory parameters, and one optional parameter
  • search_for: the first parameter; the sub-string (text) to look for
  • text_to_search: the second parameter; where to look

  • [starting_at]: optional third parameter; what character to start looking

  • Returns the position at which a string is first found within text, ignoring case.
    SEARCH("wood","How much wood can a woodchuck chuck")
    SEARCH(search_for, text_to_search, [starting_at])

Checking data types

  • Checks if data is a string (text)
    =ISTEXT("sunny weather")
    =ISTEXT(8)

  • Checks if data is a boolean (true/false) value
    =ISLOGICAL(TRUE)
    =ISLOGICAL(B12)

  • Checks if data is a number
    =ISNUMBER(136)
    =ISNUMBER(B12)

Error handling

  • Checks if a cell value is equal to an error (#N/A)
    =ISNA(A2)

  • Checks for an error in a specific cell
    =IFERROR(A1,"Error in cell A1")

=VLOOKUP(search_key, range, index, [is_sorted])

VLOOKUP looks for a value in a column; then it returns a value from any cell in the same row.

search_key: value to find
range: two or more columns, to find the data
index - the column number from the range; you can return any value in that row

Wrapping functions

  • In the multiple_functions tab
  • we can wrap multiple functions together
    =IF(NOT(ISERR(FIND( "Disney", C3, 1))), 1, 0)

  • ^ this is 3 functions in one line; starting from the innermost function

  • 1 FIND( "Disney", C3, 1)): checks the string in cell C3 to see if the substring ‚ÄúDisney‚Äù exists
  • if the substring exists, this will return the position of the substring(a number)

  • if the substring does not exist, it will return an error #VALUE!

  • 2 ISERR(value): check if the value is an error
  • in our example, it is checking if FIND( "Disney", C3, 1)) returns an error

  • wrapping this in a NOT() function (makes the next step easier)

  • 3 =IF( F3 = TRUE, 1, 0): returns 1 or 0 if the cell contains an error
  • use this value to find the total number of Disney movies

Breaking down functions

  • = to start the function. This cell is where the new value will live.

  • JOIN: the name of the function
    • usually tells you what the function will ~do~
  • (): opening and closing parentheses
    • wrap around the guts of the function
    • think of it as the container
    • contains parameters (arguement)
  • "-": the delimiter
    • the text (character) between values
  • D2,E2,F2: the values that are being joined
    • note the reference (D2) to each individual cell
    • if comma separated: individual cells
    • if colon separated: range of cells
