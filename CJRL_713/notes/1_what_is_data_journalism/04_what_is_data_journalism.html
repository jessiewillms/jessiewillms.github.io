Defining “data journalism”

“Data can be the source of data journalism, or it can be the tool with which the story is told, or it can be both. Like any source, it should be treated with skepticism; and like any tool, we should be conscious of how it can shape and restrict the stories that are created with it.”
— Paul Bradshaw, Birmingham City University

It is also:

  • Journalism that is done with (informed by) data
  • “Computer assisted reporting”
    • ie., computer programs to help election outcomes
  • A tool for reporting/storytelling.

Data journalism is NOT

  • just “interviewing” a spreadsheet
  • An beat or expertise for only a select few in a newsroom
  • Real people, experts and “word” reporting are still extremely necessary components of data journalism.

What does data journalism look like

  • Building tools to scrape/analyze data (Python)
  • Using Excel (Google Sheets, etc.) to find averages, most/least common (of a variable), identify outliers
  • Reading budgets, court documents, business filings
  • Request data/records from governments
  • Creating charts + maps to look at data
  • Making user-facing tools (ie., calculator) (JavaScript)
  • and much more

Why data journalism

  • Answers big/small questions
  • Find “hidden” stories
  • Provide context/understanding
