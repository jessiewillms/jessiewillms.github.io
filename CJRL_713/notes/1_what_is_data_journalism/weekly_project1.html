What is data journalism - Pitch a data story

Project requirements

  1. Identify 2-3 “data” resources
  2. Outline the structure of your story
  3. Light data analysis in Excel/Google Spreadsheets
  4. Create a Pivot Table

Bonus points for exploring topics we haven’t covered yet!

Getting Started

  1. Review the datasets we covered.
  2. Look at some of the data-driven stories we covered. What angels were unexplored?
  3. Consider data stories that we write on a weekly/monthly basis — housing/rental pricing data, for example.
