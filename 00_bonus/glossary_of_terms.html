Number stuff

Mean/average: If we get the sum of all values divided by the number of observations.

Finding the average test score.
Scores: 76, 87, 78, 78, 91, 12, 67
Add up all values: 76 + 87 + 78 + 78 + 91 + 12 + 67 = 489
Divide this total by number of values: 489 / 7
Average test score: 69.86

Median: If we sort the data, then find the value in the middle.

Finding the median test score.
Scores: 76, 87, 78, 78, 91, 12, 67
Scores sorted: 12, 67, 76, 78, 78, 87, 91
Middle (median) score: 78

Mode: The value that occurs most frequently in the data.

Finding the most common test score.
Scores: 76, 87, 78, 78, 91, 12, 67
Scores sorted: 12, 67, 76, 78, 78, 87, 91
Median score: 78
